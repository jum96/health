## 🚀 快速启动完整AI智能体

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置火山引擎API
复制 `.env.example` 为 `.env`，填入您的API信息：

```env
DOUBAO_API_KEY=您的火山引擎API_KEY
DOUBAO_ENDPOINT=https://ark.cn-beijing.volces.com/api/v3
DOUBAO_MODEL=您的模型ID
```

### 3. 启动智能体

**Windows用户：**
双击 `start.bat`

**或手动启动：**
```bash
python app.py
```

### 4. 访问
浏览器打开：http://localhost:5000

---

## 🛠️ 获取火山引擎API

### 步骤：
1. 登录火山引擎控制台
2. 进入"方舟（Ark）"或"大模型服务"
3. 创建模型接入点
4. 复制以下信息：
   - API Key
   - Endpoint URL
   - 模型名称（如 ep-2024...）
