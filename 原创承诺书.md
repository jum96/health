# 原创承诺书

## 创作项目名称
**岭南康养·灵境向导**

---

## 创作内容概述
本项目通过AI技术辅助，打造了一个融合岭南文化（广府、潮汕、客家）与现代健康知识的AI智能体。采用HTML/CSS/JavaScript纯前端技术，将岭南非遗（凉茶、粤剧、醒狮、龙舟、莞香）、中医药养生、二十四节气等文化元素，通过智能对话、文化科普、养生方案、节气日历等交互形式进行数字化呈现。AI主要用于辅助知识库整理、代码优化建议和内容框架设计，核心逻辑和界面设计为人工原创。

---

## 作品信息与AI工具使用声明

### 1. 作品简介
本智能体以"智创岭南・健康同行"为主题，深度融合岭南非遗文化与健康养生知识，为东莞理工学院师生及广大用户提供文化科普、健康答疑、个性化养生方案服务。创作理念是传承岭南文脉，推广健康生活方式。选取了凉茶制作技艺、粤剧、莞香、东莞龙舟等经典岭南文化元素，以及九种体质、四季养生、二十四节气等中医药文化内容，通过直观友好的界面设计，让用户在轻松互动中感受岭南文化魅力，获取实用的健康养生知识。

### 2. 使用哪些AI工具进行创作的
- Claude Code / 豆包大模型：用于代码辅助生成、知识库内容整理、设计思路建议
- AI辅助工具仅用于提供素材框架和优化建议，核心设计和代码编写为人工完成
