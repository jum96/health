## 一、智能体概述

### 1.1 名称
**岭南康养·灵境向导**

### 1.2 定位
融合岭南文化（广府、潮汕、客家）与现代健康知识的AI智能体，提供文化科普、健康答疑、个性化养生方案推荐。

### 1.3 设计思路
紧扣大赛主题「智创岭南・健康同行」，深度融合以下核心元素：
- 🏮 **岭南非遗**：凉茶制作技艺、粤剧、醒狮、龙舟竞渡、莞香
- 🎭 **粤剧曲艺**：南国红豆文化与健康养生关联
- 🌿 **中医药养生**：九种体质辨识、四季调养、岭南药膳
- 🎊 **民俗文化**：迎春花市、清明习俗、端午龙舟
- 🥟 **健康饮食**：早茶文化、老火靓汤、节气养生食谱
- 🏯 **岭南建筑**：骑楼、陈家祠等建筑文化

### 1.4 适用场景
- 校园文化科普与健康服务
- 社区健康服务站
- 文旅景点导览
- 家庭智能终端
- 在线问诊前置助手

### 1.5 服务对象
东莞理工学院师生及广大岭南文化爱好者

## 二、核心功能与交互逻辑

### 2.1 功能模块

| 功能模块 | 具体能力 | 交互示例 |
|---------|---------|---------|
| **岭南文化科普** | 讲解岭南建筑、习俗、饮食、非遗（凉茶、粤剧、醒狮、龙舟、莞香）的历史与健康智慧 | 用户问：「为什么岭南人爱喝凉茶？」→ 智能体解释湿热气候+中医清热原理，并推荐应季凉茶配方 |
| **健康知识答疑** | 基于权威医学指南+岭南地域常见病（湿热、暑病、湿困）提供解答 | 用户描述：「最近总觉得困倦，舌苔厚」→ 智能体判断湿气重，给出饮食、作息建议，并关联岭南祛湿食材（薏米、赤小豆、木棉花） |
| **养生方案推荐** | 结合节气、体质（九种体质）、岭南本地食材，生成个性化食疗/穴位按摩/导引方案 | 用户输入：年龄35，男性，痰湿体质，大暑节气 → 推荐陈皮薏米冬瓜汤、每日足三里按揉、八段锦「调理脾胃须单举」 |
| **二十四节气日历** | 可视化展示24节气，提供每个节气的岭南养生要点 | 点击「春分」节气 → 展示春分养生原则、应季食材、注意事项 |
| **九种体质自测** | 引导用户完成体质测试，获取个性化养生建议 | 点击「开始测试」→ 回答问题 → 获得体质辨识结果与养生方案 |

### 2.2 交互逻辑

```
自然语言输入（文本）
    ↓
意图识别 + 关键词匹配
    ↓
检索知识库 / 调用对应功能模块
    ↓
内容整合与模板渲染
    ↓
生成结构化回复
```

### 2.3 多轮对话
- 支持上下文关联
- 支持追问和深度探讨
- 快捷问题一键触发

## 三、AI技术框架与工具

### 3.1 技术架构

| 层级 | 技术选型 | 说明 |
|------|---------|------|
| **前端界面** | HTML5 + CSS3 + JavaScript | 纯前端实现，无需后端，可直接运行 |
| **样式设计** | 自定义CSS | 岭南风格设计，红金配色 |
| **知识库** | JavaScript对象存储 | 内置岭南非遗、养生知识、节气数据 |
| **意图识别** | 关键词匹配算法 | 本地实现，快速响应 |
| **回复生成** | 模板渲染 + 智能组合 | 预定义模板，灵活组合内容 |

### 3.2 架构图

```
┌─────────────────────────────────────────┐

├─────────────────────────────────────────┤
│         交互管理层 (JavaScript)          │
│  - 事件绑定  - 状态管理  - 路由切换    │
├─────────────────────────────────────────┤
│         智能回复层                       │
│  - 关键词匹配  - 模板选择  - 内容渲染  │
├─────────────────────────────────────────┤
│         知识库层                         │
│  - 岭南非遗  - 养生知识  - 节气数据    │
└─────────────────────────────────────────┘
```

## 四、开发工具及实现路径

### 4.1 使用工具

| 工具类型 | 具体工具 | 用途 |
|---------|---------|------|
| **开发环境** | VSCode | 代码编辑器 |
| **前端技术** | HTML5 | 页面结构 |
| | CSS3 | 样式设计、动画效果 |
| | JavaScript (ES6+) | 交互逻辑、数据管理 |
| **字体** | Google Fonts | 中文宋体 + 无衬线字体 |
| **AI辅助** | Claude Code / 豆包 | 代码辅助、内容整理 |

### 4.2 核心提示词

```
你是「岭南康养·灵境向导」AI智能体，专为东莞理工学院师生打造，核心定位是岭南文化科普、健康知识答疑、岭南养生方案推荐。

你的回答必须严格遵循以下规则：
1. 所有内容必须紧扣「岭南文化+健康」主题，融合岭南非遗、粤剧、中医药、民俗、健康饮食、岭南建筑等元素
2. 回答要专业、科学，符合岭南地域特色，针对岭南气候（湿热）、饮食习惯给出适配建议
3. 语言风格亲切易懂，适合校园师生，避免过于晦涩的专业术语
4. 当用户提问超出岭南文化/健康主题时，礼貌引导用户回到主题
5. 所有健康建议仅为科普参考，不替代专业医疗诊断，必要时提醒用户咨询专业医师
6. 结构清晰，分点输出，重点内容加粗，提升可读性
```

### 4.3 知识构建
- 使用公开文献资料整理岭南非遗、药膳方、民间验方
- 形成结构化知识库（JavaScript对象存储）
- 涵盖：岭南非遗、四季养生、九种体质、24节气、东莞特色

### 4.4 智能体框架
- 关键词匹配算法实现意图识别
- 预定义回复模板确保输出质量
- 模块化设计便于扩展

### 4.5 部署环境
- **纯前端实现**：无需后端服务器
- **运行方式**：直接用浏览器打开 `index.html`
- **兼容性**：支持主流浏览器（Chrome、Firefox、Edge、Safari）

## 五、原创性声明

### 5.1 原创性说明

本智能体系**原创作品**，具体说明如下：

| 项目 | 原创性说明 | AI生成占比 |
|------|-----------|-----------|
| **整体设计** | 功能模块、交互流程、UI风格100%人工设计 | 0% |
| **界面设计** | 配色方案、布局结构、视觉效果人工原创 | 0% |
| **代码实现** | HTML结构、CSS样式、JavaScript逻辑人工编写 | 约20%辅助 |
| **知识库内容** | 基于公开文献资料重新组织、筛选、整理 | 约30%辅助整理 |
| **核心逻辑** | 关键词匹配、回复生成算法原创设计 | 0% |

### 5.2 生成内容占比

- **人工设计与开发**：约 70%
  - 核心逻辑架构：100% 人工设计
  - 界面设计：100% 人工设计
  - 代码实现：80% 人工编写
  - 知识库内容：70% 人工整理

- **AI辅助生成**：约 30%
  - 代码辅助生成部分模板
  - 内容辅助整理素材框架
  - 提供优化建议

### 5.3 原创性声明

1. 本智能体的整体设计、交互逻辑、界面风格为**原创**
2. 知识库内容均基于公开文献资料**重新组织编写**
3. **未抄袭**现有智能体，所有代码均为**独立开发**
4. 明确标注AI技术框架及生成内容占比，符合原创要求
5. 所有引用内容均有明确来源，无知识产权争议

1. 直接用浏览器打开 `frontend/index.html`

   ```bash

   python -m http.server 8000
   ```
