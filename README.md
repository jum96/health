# 🏮 岭南康养·灵境向导

融合岭南文化与现代健康知识的AI智能体

## ✨ 项目简介

「岭南康养·灵境向导」是专为东莞理工学院师生打造的AI智能体，深度融合**岭南非遗、粤剧曲艺、中医药养生、民俗文化、健康饮食、岭南建筑**等核心元素，提供文化科普、健康答疑、个性化养生方案推荐等服务。

## 🎯 核心功能

### 1. 💬 智能对话
- 快捷问题一键提问
- 自然语言交互
- 岭南特色话题深度解答

### 2. 🏛️ 文化科普
- **非遗文化**：凉茶、粤剧、醒狮、龙舟、莞香
- **岭南建筑**：骑楼、陈家祠
- **饮食文化**：早茶、老火靓汤
- **民俗节日**：迎春花市

### 3. 🌿 养生方案
- **季节养生**：春夏秋冬四季调养
- **体质养生**：九种体质个性化建议
- **食疗药膳**：岭南特色汤品推荐

### 4. 📅 节气日历
- 二十四节气可视化展示
- 节气养生要点

### 5. 🧘 体质测试
- 九种体质自测
- 个性化养生建议

## 🚀 快速开始

### 方式一：直接打开
直接用浏览器打开 `frontend/index.html` 即可使用！

### 方式二：本地服务器
```bash
# 进入前端目录
cd frontend

# 使用 Python 启动本地服务器
python -m http.server 8000

# 或使用 Node.js
npx serve .

# 然后在浏览器访问
http://localhost:8000
```

## 📖 使用说明

### 智能对话
1. 点击左侧快捷问题，或在输入框输入问题
2. 按 Enter 发送，Shift+Enter 换行
3. 智能体会根据问题给出岭南特色回答

### 文化科普
1. 点击顶部「文化科普」标签
2. 选择分类：非遗、建筑、饮食、民俗
3. 浏览卡片内容

### 养生方案
1. 点击顶部「养生方案」标签
2. 选择类型：季节、体质、食疗
3. 查看详细养生建议

### 节气日历
1. 点击顶部「节气日历」标签
2. 浏览24节气卡片
3. 可切换年份查看

### 体质测试
1. 在对话页面点击「开始九种体质自测」
2. 回答问题完成测试
3. 获取个性化养生建议
