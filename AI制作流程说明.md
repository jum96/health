# AI制作流程说明

### 开发工具
- **前端技术**：HTML5、CSS3、JavaScript (ES6+)
- **字体**：Google Fonts (Noto Serif SC、Noto Sans SC)
- **编辑器**：VSCode

- **代码生成**：Claude Code / 豆包大模型
- **设计参考**：AI辅助界面设计
- **内容整理**：AI辅助知识库内容整理

## 三、开发流程

### 第一阶段：项目规划 (1小时)
1. 分析两份设计方案，融合创新点
2. 确定技术栈：HTML + CSS + JavaScript
3. 设计功能模块：智能对话、文化科普、养生方案、节气日历

### 第二阶段：界面设计 (2小时)
1. 设计整体UI风格，融合岭南元素（红色为主色调）
2. 实现响应式布局
3. 绘制页面原型：header、导航栏、主内容区

### 第三阶段：知识库构建 (2小时)
1. 整理岭南非遗文化知识
2. 编写四季养生、体质养生内容
3. 建立24节气数据库
4. 设计快捷问题与回答模板

### 第四阶段：核心逻辑开发 (3小时)
1. 实现标签页切换功能
2. 开发聊天对话系统
3. 实现关键词匹配回复逻辑
4. 开发文化科普、养生方案的动态渲染
5. 实现体质测试弹窗交互

### 第五阶段：测试与优化 (1小时)
1. 功能测试：检查所有模块是否正常工作
2. 兼容性测试：确保主流浏览器正常显示
3. 用户体验优化：动画效果、响应速度

## 四、生成内容占比说明

### 人工设计与开发 (约70%)
- **核心逻辑架构**：100% 人工设计（功能模块划分、交互流程）
- **界面设计**：100% 人工设计（配色方案、布局结构、UI风格）
- **知识库内容**：70% 人工整理（基于公开资料重新组织、筛选）
- **代码实现**：80% 人工编写（HTML结构、CSS样式、JavaScript逻辑）

### AI辅助生成 (约30%)
- **代码辅助**：AI辅助生成部分CSS样式、JavaScript函数模板
- **内容辅助**：AI辅助整理知识库素材，提供内容框架
- **优化建议**：AI提供用户体验优化建议

### 原创性声明
- 本智能体的整体设计、交互逻辑、界面风格为原创
- 知识库内容均基于公开文献资料重新组织编写
- 未抄袭现有智能体，所有代码均为独立开发
- 明确标注了AI技术框架及生成内容占比，符合原创要求

## 五、技术架构说明

### 前端架构
```
┌─────────────────────────────────┐
│       HTML (index.html)         │
│  - 页面结构                     │
│  - 元素布局                     │
└────────────┬────────────────────┘
             │
┌────────────▼────────────────────┐
│   CSS (css/style.css)           │
│  - 视觉样式                     │
│  - 动画效果                     │
│  - 响应式布局                   │
└────────────┬────────────────────┘
             │
┌────────────▼────────────────────┐
│  JavaScript (js/app.js)         │
│  - 交互逻辑                     │
│  - 数据管理                     │
│  - 智能回复                     │
└────────────┬────────────────────┘
             │
┌────────────▼────────────────────┐
│  知识库 (js/data.js)            │
│  - 岭南非遗                     │
│  - 养生知识                     │
│  - 节气数据                     │
└──────────────────────────────────┘
```

### 数据流程
1. 用户输入 → 意图识别 → 关键词匹配
2. 检索知识库 → 内容整合 → 模板渲染
3. 输出回复 → 界面更新 → 交互反馈

## 六、可运行演示版本说明

### 运行方式
1. 直接用浏览器打开 `frontend/index.html`
2. 或使用本地服务器：
   ```bash
   cd frontend
   python -m http.server 8000
   # 然后访问 http://localhost:8000
   ```

### 功能演示要点
1. **智能对话**：点击快捷问题或输入问题
2. **文化科普**：切换不同分类查看非遗、建筑等
3. **养生方案**：查看季节养生、体质养生、食疗药膳
4. **节气日历**：浏览24节气
5. **体质测试**：点击开始测试按钮体验流程
